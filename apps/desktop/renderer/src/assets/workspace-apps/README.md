# Workspace application icons

These SVG files are bundled with the renderer so the workspace application
launcher never depends on a CDN at runtime.

- Simple Icons assets were downloaded from `cdn.simpleicons.org` and are
  distributed under CC0 1.0.
- Devicon assets were downloaded from the Devicon repository through jsDelivr
  and are distributed under the MIT license.
